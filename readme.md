# training-single-spa

Creación de micro frontends utilizando un marco de Single-SPA

# Ejecución

cd .\single-spa-demo-root-config\
npm start -- --port 9000

cd .\single-spa-demo-header\
npm start -- --port 9001

cd .\single-spa-demo-home\  
npm start -- --port 9002

cd .\single-spa-demo-test-page\
npm start -- --port 9003

# Probar
Ingrese a: http://localhost:9000/
